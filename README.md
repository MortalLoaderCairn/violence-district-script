## ❓ FAQ

**🤔 Is there a ban risk in 2026?**
Yes — any external tool carries inherent risk. This script is designed to **reduce detection risk with reasonable use**. Avoid obvious behavior (like spectating yourself or reporting other players while using it), and stick to normal session lengths. No cheat is "100% undetectable," and we're transparent about that.

**🔄 How often is this updated?**
Frequent updates follow major Roblox patches. We typically push a new release within 24–72 hours after a game update breaks compatibility. Releases are tagged, so you'll always know when a fresh build is live.

**🛑 I'm getting "Failed to Initialize" — what do I do?**
First, make sure you're **running the executable as Administrator**. Check that your antivirus hasn't quarantined the file (add an exception for the executable if needed). Ensure you have the latest [.NET Desktop Runtime 8.0](https://dotnet.microsoft.com/en-us/download/dotnet/8.0) installed. If it persists, open an issue!

## ✨ Features

- **Auto-Aim Assist** 🎯 — Smooth target snapping with adjustable sensitivity and FOV slider. Toggle on/off with a configurable hotkey. Feels natural, not snappy.

- **Player ESP** 👁️ — See opponents through walls with customizable color-coded boxes. Distance, health, and name labels included. Frustration-free line-of-sight tracking.

- **Aimbot Smart Filter** 🔧 — Prioritize the closest target or the one with lowest health. Works best with semi-auto and full-auto weapons. Zero config required out of the box.

- **Trigger Bot** ⚡ — Automatically fires when your crosshair passes over an enemy. Adjustable delay to avoid obvious "locked-on" behavior. Keeps your finger relaxed.

- **Third-Person Camera** 📷 — Unlock the camera for a wider field of view. Frame-perfect perspective shift that respects your comfort settings. Works in vehicles too.

- **No Recoil/Spread** 🎯 — Reduces weapon bloom and vertical kick. Balanced values so your aim stays believable. Toggle per match.

- **Silent & Lightweight** 💨 — Minimal CPU/GPU overhead — around 3–5% CPU usage on a modern i5. No background services, no telemetry.

## 🛡️ Safety

This tool uses **external memory reading only** — it never writes to the Roblox process or modifies game files. Combined with reasonable in-game behavior (no spinning, no speed-hacking) and frequent updates, it offers **reduced risk for typical usage**. We recommend using alternate accounts for initial testing and avoiding streaming or recording while the tool is active.

## 🎮 How to Use

1. Launch **Roblox** and join a **Violence District** server.
2. Run the downloaded executable **as Administrator**.
3. Wait for the overlay to appear (shows "Tool Ready").
4. Press **F5** to toggle the GUI, **F6** to toggle aim assist.

| Hotkey | Action |
|--------|--------|
| F5 | Open/Close GUI |
| F6 | Toggle Aim Assist |
| F7 | Toggle ESP |
| F8 | Toggle Trigger Bot |
| F9 | Toggle No Recoil |

All keys are customizable through the GUI **Settings** tab.
